7. Part of the list

slice

The difference between copy and assignment:
assign my_foods to friend_foods instead of storing a copy of my_foods in friend_foods. This syntax actually makes Python associate the new variable friend_foods to the list contained in my_foods, so both variables point to the same list. In view of this, when we add’cannoli’ to my_foods, it will also appear in friend_foods; also, although’ice cream’ seems to have only been added to friend_foods, it will also appear in these two lists.

8. Tuples

Attempts to modify tuples are prohibited, and it is legal to assign values ​​to tuple variables
Compared to lists, tuples are simpler data structures. If a set of values ​​that need to be stored does not change throughout the life of the program, use tuples.
