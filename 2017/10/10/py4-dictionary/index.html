Dictionary

Is a series of key-value pairs. Each key is associated with a value, and you can use the key to access the value associated with it.

A key-value pair is two related values. When you specify a key, Python will return the value associated with it.
The key and value are separated by a colon, and the key-value pair is separated by a comma.

Add information

The order of the key-value pairs is different from the order of addition. Python does not care about the order in which key-value pairs are added, but only cares about the relationship between keys and values.

When using a dictionary to store user-supplied data or writing code that can automatically generate a large number of key-value pairs, you usually need to define an empty dictionary first.
