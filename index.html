Variable Name

Variable names can only contain letters, numbers and underscores. Cannot start with a number.
Variable names cannot contain spaces, but you can use underscores to separate words.

A traceback is a record that points out where the interpreter got stuck when trying to run the code.

String

All quotation marks are strings, the quotation marks can be single quotation marks’’ or double quotation marks””
title() displays each word in capital letters, that is, the first letter of each word is changed to capital.

Use the plus sign (+) to combine strings.
Tab table \t
Newline \n
Remove all blank strip()
Remove the leading blank lstrip()
Remove the blank at the end rstrip()

Number

Python uses two multiplication signs to represent power operations
When using an integer in a string, you need to explicitly indicate that you want Python to use this integer as a string. To do this, call the function str(), which tells Python to represent non-string values as strings:

In Python 2, the result of integer division contains only the integer part, and the decimal part is deleted. To avoid this, make sure that at least one operand is a floating point number, so the result will also be a floating point number
